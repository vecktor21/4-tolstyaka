Оплата

Дорогие покупатели! В нашем магазине вы можете оплатить товар любым удобным для Вас способом, а именно:

  • Оплата банковскими карточками через систему Epay Казкомерц Банка: VISA, Master Card, UPI. Разрешены транзакции по карточкам эмитированным в следующих странах: KAZ, RUS, AZE, TJK, UKR, UZB, TKM, KGZ, ARM, BLR, GEO. Не проводятся транзакции: по корпоративным картам, карточкам American Express, не поддерживающие 3D Secure (SSL) технологию. Транзакции проводятся на стороне процессингового центра Epay Казкомерц Банка
  • Через сервис по приему платежей PayBox. PayBox это универсальная система платежей с помощью которой Вы можете оплачивать товары в нашем интернет магазины используя огромное количество способов, выбирая из них наиболее удобный для Вас.
